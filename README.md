# Bài Tập Lớn: Quản lý Khách hàng & Công việc (CRM)
**Thành viên:** Phạm Duy Hòa (và nhóm)

## 1. Nguồn tham khảo & Kế thừa
- Kế thừa module Quản lý Nhân sự (HRM) từ kho `FIT-DNU/Business-Internship`.
- Sử dụng dữ liệu nhân viên từ HRM làm dữ liệu gốc để phân bổ người phụ trách.

## 2. Cải tiến cốt lõi (Mức 3 - Xuất sắc)
- Xây dựng hệ thống luồng công việc (Task) gắn liền với từng khách hàng.
- Nâng cấp giao diện Kanban kéo thả trực quan (Trello-style).
- **Tích hợp External API (Google Gemini):** Trợ lý ảo AI tự động sinh kịch bản chăm sóc khách hàng và email dựa trên ngữ cảnh công việc.

## 3. Luồng nghiệp vụ
Quy trình End-to-End mô tả sự tương tác giữa Sale, Hệ thống Odoo và Trợ lý ảo AI (Vui lòng xem file PDF trong thư mục `docs/business-flow/`).
